# Design Patterns and Principles

This folder contains my notes, examples, and practice programs on object-oriented design principles and software design patterns completed as part of the **Cognizant Digital Nurture – Deep Skilling Program**.

## Topics Covered

### Design Principles

* SOLID Principles

  * Single Responsibility Principle (SRP)
  * Open/Closed Principle (OCP)
  * Liskov Substitution Principle (LSP)
  * Interface Segregation Principle (ISP)
  * Dependency Inversion Principle (DIP)

### Creational Design Patterns

* Singleton Pattern
* Factory Method Pattern

### Structural Design Patterns

* Adapter Pattern
* Decorator Pattern
* Proxy Pattern

### Behavioral Design Patterns

* Observer Pattern
* Strategy Pattern

## Technologies Used

* Java
* Object-Oriented Programming (OOP)

## Purpose

This directory serves as a collection of my learning materials and hands-on implementations of software design principles and design patterns. The examples demonstrate best practices for writing maintainable, reusable, and scalable object-oriented applications.
